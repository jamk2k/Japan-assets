# Informeel -> Formeel

## u-werkwoorden (godan)

1. u wordt i (のむ -> のみ)
2. Voeg masu toe (のみ -> のみます)

## ru-werkwoorden (ichidan) (eindigt op **eru** of **iru**)

1. ru weglaten （たべる -> たべ)
2. masu toevoegen (たべ -> たべます)

## Onregelmatige werkwoorden (klankverandering in de stam)

* する -> します
* くる -> きます

# Formeel negatief en verleden tijd

|              | positief |    negatief    |
|:------------:|:--------:|:--------------:|
|  **heden**   |   ます   |     ません     |
| **verleden** |  ました  | ません　でした |

|              |  positief  |      negatief      |
|:------------:|:----------:|:------------------:|
|  **heden**   |  よみます  |     よみません     |
| **verleden** | よみました | よもません　でした |

# Te-vorm

## u-werkwoorden (godan)

Eindigt op:

1. う, つ of る -> って (かう -> かって)
2. む, ぶ of ぬ -> んで (のむ -> のんで)
3. く -> いて (きく -> きいて; **いく-> いって**)
4. ぐ -> いで (およぐ -> およいで)
5. す -> して (はなす -> はなして)

## ru-werkwoorden (ichidan) (eindigt op **eru** of **iru**)

Eindigt op る -> て (ねる -> ねて)

## Onregelmatige werkwoorden (klankverandering in de stam)

* する -> して
* くる -> きて
